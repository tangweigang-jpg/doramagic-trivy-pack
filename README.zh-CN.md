> 本仓库不是 `aquasecurity/trivy` 官方项目，除非另有明确说明。

语言策略：英文是 GitHub 主入口，中文是辅助入口。除非 GitHub Traffic、issue 或社区传播证明有需求，否则不维护第三语言版本。

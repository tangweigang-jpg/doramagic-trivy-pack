# Social Preview

GitHub repository social preview upload is not exposed through the stable repository contents API used by this sync script.

Use `assets/social-preview.svg` as the source asset when configuring the repository social preview through the GitHub web UI or an approved UI automation workflow.

Status fields for operations review:

- Asset path: `assets/social-preview.svg`
- Upload target: repository Settings -> Social preview
- Blocking: no
- Verification: screenshot or manual check after upload
